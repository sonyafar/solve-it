Ridders' method is a root-finding algorithm based on the false position method and the use of an exponential function to successively approximate a root of a continuous function f(x).

Given two values of the independent variable, x0 and x2, which are on two different sides of the root being sough, i.e f(x0)·f(x2)< 0.

The method begins by evaluating the function at the midpoint:

{{'$$x_1 = 0.5 \\cdot (x_0 + x_2) $$'}}

The false position method is then applied to the points x0 and x2 to computing a new value x3:

which then is used as one of the two bracketing values in the next step of the iteration.

Here we have following three different cases:

  1. If f(x3)=0 then the root is x3.
  2. If f(x1)·f(x3)<0 then root lies between x1 and x3.
  3. If f(x1)·f(x3)>0 then the next bracketing value is whichever of x0 and x2 is closer to x3 and provides the proper bracket.

So that after each iteration we redefine bracketing values and repeat the calculation process until we find the root within desired accuracy.
